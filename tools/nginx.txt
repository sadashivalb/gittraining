1. Install nginx

rpm -ivh http://nginx.org/packages/rhel/6/x86_64/RPMS/nginx-1.4.1-1.el6.ngx.x86_64.rpm

2. Update port number to 8000 in /etc/nginx/conf.d/default.conf

3. Add autoindex on;

To change the path of the file system

root /usr/bin/nginx/html

4. service nginx start

Creating the basic http Authentication for nginx .
1. Creating the htpasswd
htpasswd -c /usr/share/nginx/html/.htpasswd username Username for the http basic authentication
Then give the password for username.

2.open /etc/nginx/conf.d/default.conf Then add below lines to location / {
auth_basic "Restricted Access";
auth_basic_user_file /usr/share/nginx/html/.htpasswd;

Step 1: Generate a Private Key

openssl genrsa -des3 -out server.key 1024

Step 2: Generate a CSR (Certificate Signing Request)

openssl req -new -key server.key -out server.csr

Step 3: Remove Passphrase from Key

cp server.key server.key.org
openssl rsa -in server.key.org -out server.key

Step 4: Generating a Self-Signed Certificate

openssl x509 -req -days 365 -in server.csr -signkey server.key -out server.crt

Stepp 5: Copy the *server.crt  and server.key to the /etc/nginx/

Step 6:copy the below content to /etc/nginx/conf.d/default.conf
server section

listen 443;
ssl on;
server_name www.example.com;
ssl_certificate server.crt;
ssl_certificate_key server.key;
ssl_protocols SSLv3 TLSv1 TLSv1.1 TLSv1.2;
ssl_ciphers HIGH:!aNULL:!MD5;
listen 7000;

step 7: Restart the nginx server.
